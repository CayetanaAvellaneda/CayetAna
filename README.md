CayetAna - Ecommerce de Monos

Características
Página con scroll natural (Inicio, Productos, Nosotros, Contacto)  
Navegación sticky con scroll automático a secciones  
Carrito flotante con contador  
Proceso de checkout completo  
Integración con WhatsApp  
Diseño responsive  


Estructura de Navegación

Inicio: Sección hero con gradiente verde
Producto: Mono negro con selector de talles
Nosotros: Historia de la marca
Contacto: Botón directo a WhatsApp

Flujo de Compra

1. Cliente agrega producto al carrito (aparece automáticamente)
2. Presiona "Finalizar compra"
3. Completa datos de entrega y elige envío
4. Selecciona método de pago
5. Recibe confirmación con botón a WhatsApp
